# Natti Roohee Phani Portfolio Website

## Overview

This project is a multi-page personal portfolio website developed using modern HTML5 and CSS3. The website showcases personal information, skills, projects, and contact details in a professional and accessible format.

The portfolio follows semantic HTML5 standards, accessibility guidelines, responsive design principles, and SEO best practices.

---

## Features

* Multi-page portfolio website
* Semantic HTML5 structure
* Responsive design for mobile and desktop devices
* Accessibility-focused implementation
* SEO-friendly meta tags
* Keyboard navigable interface
* Accessible contact form
* Modern glassmorphism UI design
* Gradient color scheme and interactive effects

---

## Pages

### Home

Introduces the portfolio owner and highlights professional goals.

### About

Provides personal background, interests, and technical profile.

### Projects

Displays featured academic and personal projects.

### Contact

Contains an accessible contact form and contact information.
